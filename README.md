# React/Redux Boilerplate Application

## Tech Stack

- React
- Redux
- ES6
- Webpack
- Standard linter

- Mocha
- Enzyme
- Expect
- Expect-jsx
- Sinon

## Usefull commands

- Start application
```js
npm start
```
- Run tests (and coverage) 
```js
npm test
```
- Run tests in watch mode 
```js
npm run test-watch
```
- Run lint 
```js
npm run lint 
```
